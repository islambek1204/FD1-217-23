Аdditional homework

  • Тег dl
    создает список с подробным объяснением его элементов. Работает в связке с тегами dt и dd.
  • Тег dt
    используется при создании списка dl и создает элемент с названием термина, который будет объяснен в теге dd.
  • Тег dd
    используется при создании списка dl создает элемент с описанием термина, который будет указан в теге dt.
  • тег pre отображает предварительно отформатированный текст. Используется для отображения кода на странице.
